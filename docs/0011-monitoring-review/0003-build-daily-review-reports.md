# 0003 Build Daily Review Reports

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0011-monitoring-review/design.md`

## 目标

生成每日策略报告，帮助用户理解收益、回撤、持仓、交易和异常。

## 前置依赖

- 回测、模拟交易或实盘交易能提供持仓、交易和净值记录。

## 执行步骤

1. 汇总今日收益。
2. 汇总累计收益。
3. 计算相对基准收益。
4. 计算最大回撤。
5. 汇总当前持仓。
6. 汇总今日交易。
7. 汇总行业暴露。
8. 汇总风险事件。
9. 输出策略日志摘要。
10. 支持按日期查询日报。

## 产出

- 每日策略报告生成器。
- 报告查询接口数据。
- 策略运行复盘摘要。

## 验收

- 可以生成每日策略报告。
- 用户能看到收益、回撤、持仓、交易和异常。
- 日报结果可按日期复查。
