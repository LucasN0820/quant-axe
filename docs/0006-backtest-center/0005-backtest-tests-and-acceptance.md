# 0005 Backtest Tests And Acceptance

来源：`docs/0006-backtest-center/design.md`、`docs/0006-backtest-center/progress.md`

## 目标

用确定性样本验证回测引擎、交易约束和指标计算，确保回测结果可信且可复现。

## 前置依赖

- 回测主循环、约束和报告均已完成。

## 执行步骤

1. 构造 3 只股票、10 个交易日的小样本。
2. 固定策略信号、行情、停牌、涨跌停和成交额。
3. 测试手续费计算。
4. 测试滑点计算。
5. 测试 T+1 规则。
6. 测试停牌、涨停、跌停规则。
7. 测试同一配置重复运行结果一致。
8. 测试策略不能读取未来日期数据。
9. 测试收益、回撤和风险指标计算正确。
10. 使用真实数据跑通一个 3 年日频策略回测。

## 产出

- 回测单元测试。
- 小样本确定性测试。
- 3 年日频回测验收样例。

## 验收

- 可以跑完一个 3 年日频回测。
- 回测结果可复现。
- 回测不允许未来函数和幸存者偏差。
