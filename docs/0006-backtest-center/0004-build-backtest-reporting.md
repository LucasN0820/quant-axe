# 0004 Build Backtest Reporting

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0006-backtest-center/design.md`

## 目标

计算并提供回测报告所需的收益、回撤、风险指标、交易明细和每日持仓。

## 前置依赖

- 回测主循环和交易约束已完成。
- 每日净值、交易和持仓数据已保存。

## 执行步骤

1. 计算净值曲线。
2. 计算回撤曲线。
3. 计算年化收益。
4. 计算最大回撤。
5. 计算夏普比率。
6. 计算 Calmar 比率。
7. 计算胜率。
8. 计算换手率。
9. 计算超额收益。
10. 实现回测报告接口和分项接口。

## 产出

- 回测指标计算。
- 回测报告接口。
- 净值、回撤、持仓、交易接口。

## 验收

- 回测报告展示收益、回撤、交易明细和每日持仓。
- 指标计算可用固定样本验证。
- 报告接口能按回测任务查询。
