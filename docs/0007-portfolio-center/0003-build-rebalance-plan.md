# 0003 Build Rebalance Plan

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0007-portfolio-center/design.md`

## 目标

根据当前持仓和目标持仓生成调仓计划，输出明确买卖方向和估算金额。

## 前置依赖

- 等权组合目标持仓已完成。
- 当前持仓数据可读取。

## 执行步骤

1. 读取当前持仓权重和市值。
2. 读取目标持仓权重和目标金额。
3. 对所有当前持仓和目标持仓取并集。
4. 计算 `delta_weight` 和估算交易金额。
5. `delta_weight > 0` 生成买入计划。
6. `delta_weight < 0` 生成卖出计划。
7. 应用最小交易金额过滤，避免碎片订单。
8. 输出可供回测、模拟交易和实盘预览复用的调仓计划。

## 产出

- 调仓计划生成器。
- 最小交易金额过滤。
- 调仓差异展示数据。

## 验收

- 当前持仓和目标持仓不同时能生成正确买卖方向。
- 无差异时不生成无意义订单。
- 回测和模拟交易都能复用调仓计划。
