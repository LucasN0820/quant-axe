# 0001 Define Portfolio Models

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0007-portfolio-center/design.md`

## 目标

定义组合、当前持仓、目标持仓和调仓计划模型，让策略信号可以转化为可执行的组合结果。

## 前置依赖

- `research-lab` 已能输出策略信号。

## 执行步骤

1. 定义组合模型，包含组合 ID、名称、创建时间和配置。
2. 定义当前持仓模型，包含股票、数量、市值、权重和成本。
3. 定义目标持仓模型，包含日期、股票、目标权重和目标金额。
4. 定义调仓计划模型，包含买卖方向、当前权重、目标权重、差异权重和估算金额。
5. 设计组合输出格式，使回测和模拟交易都能复用。
6. 定义空信号、全卖出和无变化三类边界行为。

## 产出

- 组合模型。
- 当前持仓模型。
- 目标持仓模型。
- 调仓计划模型。

## 验收

- 策略信号可以生成结构化目标持仓。
- 当前持仓和目标持仓可以比较差异。
- 输出不绑定具体交易场景。
