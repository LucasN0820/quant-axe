# 0002 Build Equal Weight Portfolio

来源：`docs/0007-portfolio-center/design.md`、`docs/0007-portfolio-center/progress.md`

## 目标

实现新手阶段默认的等权组合，并支持单票上限、总仓位上限、现金保留和最大持仓数量。

## 前置依赖

- 组合模型已完成。
- 策略信号可用。

## 执行步骤

1. 从买入或持有信号中提取候选股票。
2. 应用最大持仓数量限制。
3. 按等权方式计算目标权重。
4. 应用单票最大仓位上限。
5. 应用总仓位上限。
6. 应用现金保留比例。
7. 处理权重归一化，确保总权重不超过限制。
8. 默认方案采用 10 只股票、95% 总仓位、5% 现金保留，待确认后可调整。

## 产出

- 等权组合构建器。
- 仓位约束处理。
- 默认新手组合配置。

## 验收

- 给定 10 个买入信号，等权组合每只目标约 10%。
- 单票和总仓位上限生效。
- 空信号时返回空目标仓位。
