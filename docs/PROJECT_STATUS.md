# QuantDash 项目进度

更新时间：2026-05-03

## 已完成

### 项目结构与工程化

- 前后端已拆分为平级目录：`frontend/` 与 `backend/`。
- 根目录引入 `Taskfile.yml`，支持一条命令并行启动前后端。
- 后端依赖使用项目内 `.venv`，避免污染系统 Python。
- 前端依赖位于 `frontend/node_modules`。
- `.gitignore` 已覆盖 `.venv`、`node_modules`、`.next`、`__pycache__` 等本地产物。
- 后端引入 Pylint，并纳入 `task check`。
- `task check` 已覆盖：后端语法检查、后端 Pylint、前端 lint、前端 build。

### 前端

- 使用 Next.js App Router 构建看板页面。
- UI 已完成基础三栏工作台布局：
  - 左侧自选股与搜索。
  - 顶部市场指数。
  - 中部 K 线图与新闻/盘口区域。
  - 右侧财务估值与舆情区域。
- 引入 Zustand 管理复杂状态：
  - 搜索输入。
  - 自选股列表。
  - 当前选中股票。
  - K 线周期。
  - quote 缓存。
  - 搜索查询状态。
  - 指数数据。
  - K 线数据。
- 自选股支持 `localStorage` 本地持久化。
- 自选股搜索支持：
  - 输入代码、名称、拼音时显示本地候选。
  - 输入完整 6 位股票代码时查询真实 quote 并添加。
- K 线图使用 ECharts：
  - 支持日线、周线、月线、年线。
  - 支持 MA5、MA10。
  - 支持成交量。
  - 支持底部日期范围滑块。
  - 支持鼠标滚轮/触控板缩放查看历史。
- 涨跌颜色已调整为：上涨绿色，下跌红色。

### 后端与真实数据

- 使用 FastAPI 提供行情数据服务。
- Next.js Route Handlers 作为 BFF 层请求 FastAPI 后端。
- 当前真实行情源为 Sina A 股公开接口。
- 已实现接口：
  - `GET /health`
  - `GET /api/market/indexes`
  - `GET /api/stock/quote/{symbol}`
  - `GET /api/stock/kline/{symbol}?type=daily|weekly|monthly|yearly`
- 已接入真实数据：
  - 个股 quote：现价、涨跌幅、成交量、成交额、最高/最低、开盘、昨收。
  - 大盘指数：上证指数、深证成指、创业板指、科创 50。
  - 个股 K 线：日线、周线、月线、年线。
- K 线历史范围支持近 10 年。
- 周线、月线、年线由后端基于日 K 聚合生成。
