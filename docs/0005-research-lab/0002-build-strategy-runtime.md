# 0002 Build Strategy Runtime

来源：`docs/0005-research-lab/design.md`、`docs/0005-research-lab/progress.md`

## 目标

建立策略上下文加载器和 Python 策略插件接口，让策略可以在不依赖回测引擎的情况下生成信号。

## 前置依赖

- 策略和信号模型已完成。
- 股票池和因子接口可用。

## 执行步骤

1. 实现策略上下文加载器，输入策略、日期、股票池、因子和当前持仓。
2. 定义 Python 策略插件目录和加载规则。
3. 约定所有策略暴露 `generate_signals(context) -> list[Signal]`。
4. 为策略插件提供只读数据访问，禁止直接访问未来日期。
5. 实现策略执行错误捕获和日志记录。
6. 实现信号预览接口，不依赖回测引擎即可运行。
7. 实现信号结果持久化。
8. 增加未来函数检测测试。

## 产出

- 策略上下文加载器。
- Python 策略插件接口。
- 信号预览和持久化能力。

## 验收

- 策略能在指定日期输出买入、卖出或持有信号。
- 策略 preview 不需要启动回测。
- 策略不能读取未来日期数据。
