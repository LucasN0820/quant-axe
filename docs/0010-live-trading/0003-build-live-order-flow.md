# 0003 Build Live Order Flow

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0010-live-trading/design.md`

## 目标

实现账户同步、持仓同步、订单预览、风控检查、人工确认、撤单、成交回报和紧急停止。

## 前置依赖

- mock broker adapter 已完成。
- 风控接口可用。
- 组合调仓计划可用。

## 执行步骤

1. 实现实盘账户资金同步。
2. 实现实盘持仓同步。
3. 实现订单预览，预估价格、金额和风险结果。
4. 订单预览前调用风控检查。
5. 风控失败时不能生成可确认订单。
6. 实现人工确认后下单。
7. 未人工确认的订单不能下发。
8. 实现撤单。
9. 实现成交回报同步。
10. 实现紧急停止，开启后所有新订单被拒绝。

## 产出

- 实盘账户和持仓同步。
- 实盘订单预览和确认流程。
- 撤单、成交回报和紧急停止。

## 验收

- 实盘订单下发前必须通过风控和人工确认。
- 紧急停止能够阻止后续下单。
- 券商返回失败时记录审计日志。
