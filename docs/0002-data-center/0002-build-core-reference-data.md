# 0002 Build Core Reference Data

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0002-data-center/design.md`

## 目标

建设股票基础信息、交易日历、ST、停牌和涨跌停等核心参考数据，支撑股票池、回测和风控。

## 前置依赖

- 数据源 adapter 已建立。
- PostgreSQL 存储可用。

## 执行步骤

1. 建立 `stock_profiles`，包含代码、名称、交易所、行业、上市日期、退市日期和拼音字段。
2. 建立 `trade_calendar`，记录交易所、日期和是否开市。
3. 建立 `stock_status`，记录每日 ST 和停牌状态。
4. 建立 `limit_prices`，记录每日涨停价和跌停价。
5. 为所有核心表补齐 `source`、`updated_at` 或等价可追溯字段。
6. 编写每日更新任务，分别更新股票基础信息、交易日历和交易状态。
7. 对退市、停牌、ST 状态变化保留历史记录，避免回测使用当前状态替代历史状态。

## 产出

- 股票基础信息表。
- 交易日历表。
- ST、停牌、涨跌停数据表。
- 基础数据更新任务。

## 验收

- 可以判断某只股票某日是否 ST、停牌、涨停、跌停。
- 股票基础信息可按代码和名称查询。
- 历史状态不会被当前状态覆盖。
