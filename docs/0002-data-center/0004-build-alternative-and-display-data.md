# 0004 Build Alternative And Display Data

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0002-data-center/design.md`

## 目标

补齐行情看板和后续情绪、价值因子所需的财务、新闻、公告、舆情、分时、盘口和逐笔成交数据。

## 前置依赖

- 数据源 adapter 已建立。
- 核心股票资料可查询。

## 执行步骤

1. 建立 `financial_metrics`，记录 PE TTM、PB、ROE、毛利率、报告期和来源。
2. 建立 `news_items`，区分新闻、公告、财报、回购、减持、增持等类型。
3. 建立 `hot_keywords`，记录热词、热度、来源和采集时间。
4. 接入真实分时图数据接口。
5. 接入真实五档盘口数据接口。
6. 接入真实逐笔成交数据接口。
7. 为新闻、公告、财务和舆情任务设计分钟、小时或财报周期更新频率。
8. 对数据缺失保留空结果，不生成投资结论。

## 产出

- 财务指标数据。
- 新闻公告数据。
- 舆情热词数据。
- 分时、盘口、逐笔成交接口。

## 验收

- 行情看板可以移除主要 mock 新闻、盘口、舆情和财务数据。
- 财务和舆情数据包含来源与更新时间。
- 数据缺失不会导致接口异常。
