# 0002 Build Position And Exposure Rules

来源：`docs/0008-risk-center/design.md`、`docs/0008-risk-center/progress.md`

## 目标

实现组合和仓位相关风控规则，防止过高集中度和超额暴露。

## 前置依赖

- 风控模型已完成。
- 当前持仓、目标持仓和订单快照可用。

## 执行步骤

1. 实现单票仓位限制，超过上限则阻塞。
2. 实现行业仓位限制，超过上限则阻塞或警告。
3. 实现总仓位限制，超过总暴露则阻塞。
4. 实现最大订单金额限制。
5. 实现最大回撤规则，超过阈值暂停策略或降仓。
6. 实现连续亏损降仓规则。
7. 每条规则输出独立原因，避免只返回笼统失败。

## 产出

- 仓位类风控规则。
- 暴露类风控规则。
- 回撤和连续亏损规则。

## 验收

- 超过单票仓位上限的订单能被拦截。
- 组合总仓位超限时输出明确原因。
- 回撤规则可根据策略或组合状态触发。
