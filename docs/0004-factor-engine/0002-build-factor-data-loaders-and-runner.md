# 0002 Build Factor Data Loaders And Runner

来源：`docs/0004-factor-engine/design.md`、`docs/0004-factor-engine/progress.md`

## 目标

建立因子计算的数据读取器和批量计算任务，保证因子只使用当前日期及以前可获得的数据。

## 前置依赖

- 因子模型和存储已完成。
- `data-center` 提供历史行情和财务披露数据。
- `universe-center` 提供指定日期股票池。

## 执行步骤

1. 实现行情数据读取器，支持按 symbol、日期范围和字段读取。
2. 实现股票池读取器，支持按 `universe_id + date` 读取历史成分。
3. 实现财务数据读取器，必须考虑财报披露日期。
4. 实现舆情数据读取器，按采集时间避免未来数据。
5. 实现因子计算上下文，包含当前日期、股票池、输入数据和因子版本。
6. 实现批量计算任务，支持单日期、日期区间和指定股票池。
7. 支持增量更新，只计算新增日期或缺失记录。
8. 为任务记录状态、错误和耗时。

## 产出

- 行情、股票池、财务、舆情读取器。
- 因子计算上下文。
- 因子批量计算任务。

## 验收

- 因子计算不会读取未来日期数据。
- 批量任务失败时能定位具体因子和日期。
- 可以按某日某股票池触发因子计算。
