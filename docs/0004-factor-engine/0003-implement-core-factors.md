# 0003 Implement Core Factors

来源：`docs/proposal.md`、`docs/total_progress.md`、`docs/0004-factor-engine/design.md`

## 目标

实现第一批新手优先的基础因子，支撑 20 日动量策略和后续基础策略研究。

## 前置依赖

- 因子计算 runner 已完成。
- 历史行情、成交额和股票池快照可用。

## 执行步骤

1. 实现 20 日动量因子。
2. 实现 60 日动量因子。
3. 实现 1 日反转因子。
4. 实现成交额均值因子。
5. 实现 20 日波动率因子。
6. 为每个因子声明输入字段、计算窗口、频率和版本。
7. 使用固定行情样本测试 20 日动量和 60 日动量。
8. 使用固定成交额样本测试流动性因子。
9. 使用缺失值样本测试空值处理。

## 产出

- P0 因子实现。
- 因子定义注册。
- 因子计算单元测试。

## 验收

- 可以计算某日某股票池的 20 日动量。
- 缺失历史窗口时结果为 `null` 或明确状态。
- 因子结果可保存并按日期查询。
